# LLaVA Vision Querying

Esta variante ejecuta la practica 14 con un modelo de vision real en `Ollama` usando `llava`. Usa una imagen local de una escena urbana y hace una consulta de descripcion y conteo de coches.

## Requisitos

1. Instalar `Ollama`.
2. Ejecutar `ollama serve`.
3. Descargar el modelo: `ollama pull llava`.

## Ejecucion

```powershell
.\venv\Scripts\python.exe .\spikes\14-basic_vision_multimodal_lab\llava_vision_querying\main.py
```
