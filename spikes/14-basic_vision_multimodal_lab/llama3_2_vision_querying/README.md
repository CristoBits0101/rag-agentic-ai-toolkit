# Llama3.2 Vision Querying

Esta variante ejecuta la practica 14 con un modelo de vision real en `Ollama` usando `llama3.2-vision`. Usa la misma escena urbana local y consulta descripcion general y conteo de coches.

## Requisitos

1. Instalar `Ollama`.
2. Ejecutar `ollama serve`.
3. Descargar el modelo: `ollama pull llama3.2-vision`.

## Ejecucion

```powershell
.\venv\Scripts\python.exe .\spikes\14-basic_vision_multimodal_lab\llama3_2_vision_querying\main.py
```
