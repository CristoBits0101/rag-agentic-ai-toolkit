# Style Finder con Llama 3.2 Vision

Esta variante del Style Finder usa `llama3.2-vision` como modelo principal para el analisis de moda.

```powershell
ollama serve
ollama pull llama3.2-vision
.\venv\Scripts\python.exe .\spikes\14-basic_vision_multimodal_lab\style_finder_llama3_2_vision_app\main.py
```
