# DALL-E 2 Generation

Esta variante ejecuta la practica 16 con `dall-e-2` usando el prompt `a white siamese cat` y guarda la imagen en local.

## Requisitos

1. Instalar `openai`: `pip install -U openai`.
2. Exportar `OPENAI_API_KEY`.

## Ejecucion

```powershell
.\venv\Scripts\python.exe .\spikes\16-dalle_image_generation_lab\dall_e_2_generation\main.py
```
